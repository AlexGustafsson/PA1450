Visualizations

Warming Stripes

Warming Stripes display the temperature over a period of time as a stripe colored depending on the temperature.

For more information see https://en.wikipedia.org/wiki/Warming_stripes

The API can be used over at /visualization/warming-stripes. There are two required parameters, width and height for controlling the size of the resulting visualization.
